different git commends and their use
  • git init - making git start tracking changes
  • git status - look at the status of the git in that branch
  • git add - need to do this before starting to track, basically adding this to the tracking system
  • git commit -m "new file" - officially commiting the file (after adding it to the tracking system) and need to add a note that indicates what changes were made
  • git remote -v - this will connect locally created files (that was added and commit) to the only github repository
  • git push - this is pushing (uploading) what was committed locally to the github

    3. HTML semantic elements and its significance
    Semantic elements are “tags” in the HTML code that have meaningful names to the browser and the developer. The semantic tags in HTML5 can help increase the accessibility of the webpage to the assistive technologies and help the computers better interpret the content. It also helps internationalize the HTML5 tags for the non-English speakers and other programmers when they try to look at the structure of the webpage created. The semantic tags allow the HTML coding information to be shared across different applications and programmers.
